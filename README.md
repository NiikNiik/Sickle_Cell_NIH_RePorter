# Sickle_Cell_NIH_RePorter

This repository explores NIH RePORTER awards related to the Cure Sickle Cell Initiative. The `Sickle_Cell_initiative.ipynb` notebook cleans the data, visualizes funding patterns, and applies lightweight NLP for topic discovery and label prediction.

## Quick review

* Notebook (interactive): [`Sickle_Cell_initiative.ipynb`](Sickle_Cell_initiative.ipynb)
* HTML preview (no execution required): [`Sickle_Cell_initiative.html`](Sickle_Cell_initiative.html)

Open the HTML file to skim the narrative, code, and chart descriptions. Run the notebook locally to regenerate the figures and metrics.
