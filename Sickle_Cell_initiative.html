NIH Sickle Cell Analysis (Quick Preview)

This HTML is a light preview of the notebook. Open Sickle_Cell_initiative.ipynb to run the code and see the charts.

Data

  • SickleCell_Gene_Allyrs.csv (sickle cell grants)
  • NHLBI_Cooleys_Anemia.csv (Cooley's anemia grants)

Both files are loaded, cleaned, and combined with a simple dataset label so their funding can be compared.

Key Charts

  1. Top-funded institutions: A bar chart of the 10 organizations receiving the most sickle cell dollars.
  2. Funding by fiscal year: Lines showing how sickle cell spending changes over time.
  3. Sickle cell vs. Cooley's anemia: Side-by-side bars and yearly lines to compare total costs across the two files.

Simple Machine Learning

A basic bag-of-words logistic regression predicts whether a project comes from the sickle cell or Cooley's anemia file. The notebook prints accuracy and a confusion matrix to keep it easy to read.

Takeaways

  • Funding by institution and year stays front and center.
  • The two-condition comparison shows where dollars lean.
  • The quick text model is a gentle intro to ML, not an advanced pipeline.
